> An autonomous service robot built with ROS 2 Humble, YOLOv8 deep learning, and Sliding Mode Control. The robot detects people in a simulated cafe environment using real-time object detection, then autonomously approaches them while avoiding obstacles using a nonlinear SMC controller — completing perception-driven navigation with zero human intervention.
